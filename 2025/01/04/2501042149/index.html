定义1.2.4 以集合为元素的集合称为集族

设A1,A2,A3\large A_1,A_2,A_3A1​,A2​,A3​为集合，则{A1,A2,A3}\large \left \{ A_1,A_2,A_3 \right \}{A1​,A2​,A3​}为一个集族。若令I={1,2,3}\large I=\left \{ 1,2,3 \right \}I={1,2,3}，则∀i∈I\large \forall i\in I∀i∈I，i\large ii确定了一个唯一的集合Ai\large A_iAi​

集族{A1,A2,A3}\large \left \{ A_1,A_2,A_3 \right \}{A1​,A2​,A3​}又常写成{Ai}i∈I\large \left \{ A_i \right \} _{i\in I}{Ai​}i∈I​，即I\large II中元素i\large ii确定的那些集形成的集族

定义1.2.5

集合S\large SS的所有子集(包括空集∅\large \varnothing∅及S\large SS本身)形成的集族称为S\large SS的幂集，并记为2S\large 2^{S}2S，或P(S)\large \mathscr{P}(S)P(S)

(P\large \mathscr{P}P是字母P\large PP)

于是，2S={A∣A⊆S}\large 2^{S}=\left \{ A | A\subseteq S \right \}2S={A∣A⊆S}

例: 设S={1,2,3}\large S= \left \{ 1,2,3 \right \}S={1,2,3}，则2S={∅,{1}{2}{3}{1,2}{1,3}{2,3}{1,2,3}}\large 2^{S}=\left \{\varnothing ,\left \{ 1 \right \} \left \{ 2 \right \} \left \{ 3 \right \} \left \{ 1,2 \right \} \left \{ 1,3 \right \} \left \{ 2,3 \right \} \left \{ 1,2,3 \right \} \right \}2S={∅,{1}{2}{3}{1,2}{1,3}{2,3}{1,2,3}}

S\large SS有八个子集

区分∅\large \varnothing∅和{∅}\large \left \{ \varnothing \right \}{∅}

2∅={∅}\large 2^{\varnothing}=\left \{ \varnothing \right \}2∅={∅}

∅\large \varnothing∅为空集，而{∅}\large \left \{ \varnothing \right \}{∅}是一个集族，这个集族仅有一个元素，就是空集。

因此，∅≠{∅}\large \varnothing \neq \left \{ \varnothing \right \}∅​={∅}，但∅∈{∅}\large \varnothing \in \left \{ \varnothing \right \}∅∈{∅}且∅⊆{∅}\large \varnothing \subseteq \left \{ \varnothing \right \}∅⊆{∅} (因为空集是任何集合的子集)。集{∅,{∅}}\left \{ \varnothing , \left \{ \varnothing \right \} \right \}{∅,{∅}}含有两个元素。

多个集合的并集

A1∪A2∪⋯∪An\large A_1 \cup A_2 \cup \cdots \cup A_nA1​∪A2​∪⋯∪An​缩写为⋃i=1nAi\large \bigcup\limits_{i=1}^{n}A_ii=1⋃n​Ai​

若A1,A2,⋯ ,An,⋯\large A_1,A_2,\cdots ,A_n,\cdotsA1​,A2​,⋯,An​,⋯是一个集合的无穷序列，则它们的并集缩写为⋃n=1∞An\large \bigcup\limits_{n=1}^{\infty}A_nn=1⋃∞​An​其定义为⋃n=1∞An={x∣∃n∈N 使得 x∈An}\large \bigcup\limits_{n=1}^{\infty}A_n=\left \{ x | \exists n \in N\text{ 使得 }x\in A_{n} \right \}n=1⋃∞​An​={x∣∃n∈N 使得 x∈An​}，N\large NN是自然数之集

若{Al}l∈I\large \left \{ A_l \right \} _{l\in I}{Al​}l∈I​是任一集族，则集族中那些集之并集记为⋃l=IAl\large \bigcup\limits_{l=I}A_ll=I⋃​Al​ 并且⋃l=IAl={x∣∃l∈I 使得 x∈Al}\large \bigcup\limits_{l=I}A_l= \left \{ x | \exists l \in I \text{ 使得 }x\in A_{l} \right \}l=I⋃​Al​={x∣∃l∈I 使得 x∈Al​}

多个集合的交集

A1∩A2∩⋯∩An=⋂i=1nAn={x∣∀i∈{1,2,⋯ ,n},x∈Ai}\large A_1 \cap A_2 \cap \cdots \cap A_n=\bigcap\limits_{i=1}^{n}A_n=\left \{ x | \forall i \in \left \{ 1,2,\cdots ,n \right \} ,x\in A_{i} \right \}A1​∩A2​∩⋯∩An​=i=1⋂n​An​={x∣∀i∈{1,2,⋯,n},x∈Ai​}

A1∩A2∩⋯∩An∩⋯=⋂n=1∞An={x∣∀n∈N,x∈An}\large A_1 \cap A_2 \cap \cdots \cap A_n \cap \cdots=\bigcap\limits_{n=1}^{\infty}A_n=\left \{ x | \forall n \in N ,x\in A_{n} \right \}A1​∩A2​∩⋯∩An​∩⋯=n=1⋂∞​An​={x∣∀n∈N,x∈An​}

集族{Al}l∈I\large \left \{ A_l \right \} _{l\in I}{Al​}l∈I​各集的交记为⋂l=IAl={x∣∀ξ∈I 使得 x∈Aξ}\large \bigcap\limits_{l=I}A_l=\left \{ x | \forall \xi \in I\text{ 使得 }x\in A_{\xi} \right \}l=I⋂​Al​={x∣∀ξ∈I 使得 x∈Aξ​}

定理1.3.3

设A\large AA为任一集合，{Bl}l∈I\large \left \{ B_l \right \} _{l\in I}{Bl​}l∈I​为任一集族，则

A∩(⋃l=IBl)=⋃l=I(A∩Bl)A∪(⋂l=IBl)=⋂l=I(A∪Bl)\begin{aligned} \large A \cap \left ( \bigcup\limits_{l=I}B_l \right ) &=\large \bigcup\limits_{l=I} \left ( A\cap B_{l} \right ) \\\\ \large A \cup \left ( \bigcap\limits_{l=I}B_l \right ) &=\large \bigcap\limits_{l=I} \left ( A\cup B_{l} \right ) \end{aligned} A∩⎝⎜⎛​l=I⋃​Bl​⎠⎟⎞​A∪⎝⎜⎛​l=I⋂​Bl​⎠⎟⎞​​=l=I⋃​(A∩Bl​)=l=I⋂​(A∪Bl​)​

其中I≠∅\large I \neq \varnothingI​=∅

交并运算满足分配律

A∩(B∪C)=(A∩B)∪(A∩C)A∪(B∩C)=(A∪B)∩(A∪C)\begin{aligned} \large A \cap \left ( B \cup C \right )&=\large \left ( A \cap B \right ) \cup \left ( A \cap C \right ) \\\\ \large A \cup \left ( B \cap C \right )&=\large \left ( A \cup B \right ) \cap \left ( A \cup C \right ) \end{aligned} A∩(B∪C)A∪(B∩C)​=(A∩B)∪(A∩C)=(A∪B)∩(A∪C)​

定理 1.3.5 吸收律

A∩(A∪B)=AA∪(A∩B)=A\begin{aligned} \large A \cap \left ( A \cup B \right )&=\large A \\\\ \large A \cup \left ( A \cap B \right )&=\large A \end{aligned} A∩(A∪B)A∪(A∩B)​=A=A​

差集 定义 1.3.4

设A\large AA与B\large BB为两个任意的集合，由属于A\large AA但不属于B\large BB的一切元素构成的集合称为A\large AA与B\large BB的差集，并记为A\B\large A \backslash BA\B

A\B={x∣x∈A 且 x∉B}\large A \backslash B=\left \{ x | x \in A \text{ 且 }x \notin B \right \}A\B={x∣x∈A 且 x∈/​B}

A\large AA有B\large BB没有的元素

差集分配律 定理 1.3.6

设A,B,C\large A,B,CA,B,C为任意三个集合，则A∩(B\C)=(A∩B)\(A∩C)\large A\cap \left ( B \backslash C \right )=\left (A\cap B \right ) \backslash \left ( A\cap C \right )A∩(B\C)=(A∩B)\(A∩C)

差集不满足交换律

A\B≠B\A\large A\backslash B \neq B \backslash AA\B​=B\A

定理 1.3.7

(A\B)∪B=A⇔B⊆A\large \left ( A\backslash B \right )\cup B=A\Leftrightarrow B\subseteq A(A\B)∪B=A⇔B⊆A

对称差

定义1.3.5 A\B\large A\backslash BA\B与B\A\large B\backslash AB\A的并集称为A\large AA与B\large BB的对称差，记为AΔB\large A \Delta BAΔB

AΔB=(A\B)∪(B\A)={x∣x∈A∪B 且 x∉A∩B}={x∣x∈A 或 x∈B 但 x∉A∩B}\large A \Delta B=\left ( A\backslash B\right )\cup \left ( B\backslash A \right )=\left \{ x | x\in A\cup B \text{ 且 }x\notin A\cap B \right \}=\left \{ x | x\in A \text{ 或 }x\in B \text{ 但 }x\notin A\cap B \right \}AΔB=(A\B)∪(B\A)={x∣x∈A∪B 且 x∈/​A∩B}={x∣x∈A 或 x∈B 但 x∈/​A∩B}

说白了就是A\large AA和B\large BB不同时含有的元素

对称差性质 定理 1.3.8

AΔB=BΔA 满足交换律(AΔB)ΔC=AΔ(BΔC) 满足结合律AΔA=∅AΔ∅=A\begin{aligned} &\large A \Delta B =\large B \Delta A \text{ 满足交换律} \\ &\large \left ( A \Delta B \right )\Delta C=\large A \Delta \left ( B \Delta C \right ) \text{ 满足结合律} \\ &\large A \Delta A=\varnothing \\ &\large A \Delta \varnothing = A \end{aligned} ​AΔB=BΔA 满足交换律(AΔB)ΔC=AΔ(BΔC) 满足结合律AΔA=∅AΔ∅=A​

交运算关于对称差满足分配律

A∩(BΔC)=(A∩B)Δ(A∩C)\large A \cap \left ( B \Delta C \right )=\left ( A \cap B \right )\Delta\left ( A \cap C \right )A∩(BΔC)=(A∩B)Δ(A∩C)

余集 定义 1.4.1

设S\large SS是一个集合，A⊆S\large A\subseteq SA⊆S，差集S\A\large S\backslash AS\A称为集A\large AA对集S\large SS的余集，记为Ac\large A^{c}Ac，即Ac=S\A\large A^{c}=S\backslash AAc=S\A

余集也称为补集，有的使用CSA\large C_{S}ACS​A表示A\large AA对S\large SS的余集(本人高中就学的这种写法)

性质

S\large SS对S\large SS的余集Sc\large S^{c}Sc为空集,即

CSS=Sc=∅\large C_{S}S=S^{c}=\varnothingCS​S=Sc=∅

∅c=S(CS∅=S)\large \varnothing^{c}=S\left ( C_{S}\varnothing =S \right )∅c=S(CS​∅=S)

A∩Ac=∅\large A\cap A^{c}=\varnothingA∩Ac=∅，即CSA∩A=∅\large C_{S}A\cap A=\varnothingCS​A∩A=∅

A∪Ac=S\large A\cup A^{c}=SA∪Ac=S，即A∪CSA=S\large A\cup C_{S}A=SA∪CS​A=S

定理 1.4.1

并集的余集等于各余集的交集 (⋃ξ∈IAξ)c=⋂ξ∈IAξc\large \left ( \bigcup\limits_{\xi \in I}A_{\xi} \right )^{c}=\bigcap\limits_{\xi \in I}A_{\xi}^{c}(ξ∈I⋃​Aξ​)c=ξ∈I⋂​Aξc​

定理 1.4.2

交集的余集等于各余集的并集 (⋂ξ∈IAξ)c=⋃ξ∈IAξc\large \left ( \bigcap\limits_{\xi \in I}A_{\xi} \right )^{c}=\bigcup\limits_{\xi \in I}A_{\xi}^{c}(ξ∈I⋂​Aξ​)c=ξ∈I⋃​Aξc​

定理1.4.1和1.4.2称为De Morgan\large \text{De Morgan}De Morgan公式。在有限的形式下，我们有公式

(A∪B)c=Ac∩Bc\large \left ( A\cup B \right )^{c}=A^{c}\cap B^{c}(A∪B)c=Ac∩Bc

(A∩B)c=Ac∪Bc\large \left ( A\cap B \right )^{c}=A^{c}\cup B^{c}(A∩B)c=Ac∪Bc

余集，差集，对称差之间的联系 定理 1.4.3

设A\large AA和B\large BB都是S\large SS的子集，则

A\B=A∩Bc\large A\backslash B=A\cap B^{c}A\B=A∩Bc

AΔB=(A∩Bc)∪(B∩Ac)\large A\Delta B=\left( A\cap B^{c} \right )\cup \left ( B\cap A^{c} \right )AΔB=(A∩Bc)∪(B∩Ac)

Ac=SΔA\large A^{c}=S\Delta AAc=SΔA

对偶原理

若有关集的并、交及余集运算的某一关系式成立，如果将式中的记号∪,∩,⊆,⊇\large \cup ,\cap ,\subseteq ,\supseteq∪,∩,⊆,⊇分别换成∩,∪,⊇,⊆\large \cap ,\cup ,\supseteq ,\subseteq∩,∪,⊇,⊆，等号保持不变，并将式中每个集换成它们的余集，由此得到的关系式一定成立
